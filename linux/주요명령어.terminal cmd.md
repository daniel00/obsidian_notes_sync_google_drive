#### 리눅스, 유닉스에서 사용되는 주요 명령어를 정리합니다.


##### grep
문자열 검색
grep test ~/.bashrc      test 문자열 검색
grep -i test ~/.bashrc   대소문자 구분 안함
grep -v test ~/.bashrc  test문자열이 포함되지 않은 행만 검색
grep -r test ~/study     study의 하위디렉토리까지 검색
grep -l test ~/study     해당 문자열을 포함한 파일을 보여줌
