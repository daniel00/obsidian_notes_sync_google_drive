#### 개발환경

MCU : STM32F103C8T6
다운로더: ST LINK V2

오른쪽 핀헤더 사용 : ST-LINK V2

SWCLK
SWDIO
GND
3.3V <---- 이 보드는 3.3V 사용
5.0V


![[Pasted image 20260317214426.png]]



내 우분투에 연결한 상태는 아래와 같다.
- stm32에는 마이크로USB커넥터를 통해 전원을 인가한다.
- st-link v2 는 USB로 우분투머신에 연결하고, 커넥터는 stm32의 st-link v2핀헤더에 연결한다.

![[20260317_221639.jpg]]

##### ST LINK V2를 리눅스에서 인식 시키는 법

###### rules 파일 생성
sudo vim /etc/udev/rules.d/49-stlinkv2.rules

###### 아래 내용 입력 & 저장

```text
# ST-LINK/V2
SUBSYSTEMS=="usb", ATTRS{idVendor}=="0483", ATTRS{idProduct}=="3748", \
    MODE:="0666", \
    SYMLINK+="stlinkv2_%n"

# ST-LINK/V2-1
SUBSYSTEMS=="usb", ATTRS{idVendor}=="0483", ATTRS{idProduct}=="374b", \
    MODE:="0666", \
    SYMLINK+="stlinkv2-1_%n"
```


###### 설정 적용하기
 
 ```bash
sudo udevadm control --reload-rules
sudo udevadm trigger
 ```

###### ST LINK V2 연결

###### 아래 명령어로 연결 되었는지 확인
```bash
lsusb | grep -i stm
```
