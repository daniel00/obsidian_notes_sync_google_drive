#### flet앱을 안드로이드에서 실행하는 방법에 대해 설명 합니다.

- 안드로이드 디바이스에 "flet" 앱 설치-실행
- flet run --android 로 실행
- 터미널에 표시된 QR코드를 카메라 앱 또는 QR리더기로 스캔
- 링크를 클릭
- 앱 실행 됨!!!
- 앱 화면을 3손가락으로 누르거나 , 흔들기를 하면 앱이 종료되고 flet 프로젝트 화면으로 이동한다.
