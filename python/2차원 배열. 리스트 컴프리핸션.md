#### 파이썬에서 2차원 배열. 사용 하는 방법

각 row당 크기가  cols 개인 리스트를 하나씩 가지고 있다고 생각하면 된다.
row1 - col1~coln
row2 - col1~coln
...
rown - col1~col


1. 리스트 컴프리핸션 방법

```python
rows, cols = 3,3
array = [[0] * cols for _ in rnage(rows)]
```

위 표현은 다음과 같다.

```python
array = [[0 for _ in range(cols)] for _in range(rows)]
```
