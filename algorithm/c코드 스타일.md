K&R, BSD(Allman스타일 이라고도 부른다) 등의 스타일이 있다.

나는 BSD(Allman) 스타일이 좋다.
라인수는 많아지지만, 읽기가 훨씬 편하다.


BSD
```cpp
int main()
{ //여기서 괄호를 연다
	printf("hello world");
}
```

K&R
```cpp
int main(){ //여기서 괄호를 연다
	printf("hello world");
}
```
