### i3 wm은 리눅스용 윈도우 메니저이다.
config 파일은 ~/.config/i3/config 파일이고,
나는 mod 키로 window key를 지정하였으며,
앱 실행은, mod + d를 사용해서 런처가 뜨면 타이핑해서 원하는 앱을 열면 된다.
터미널 열기 mod + enter 
윈도우 포커스 이동 mod + arrow key
윈도우 닫기 mod + shift + q

config파일은 따로 정리해야겠다.
